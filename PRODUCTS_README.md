# 产品知识库

## 使用方式

编辑 `products.json` 文件，按商品 ID 添加产品信息。

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| `title` | 商品标题 | iPhone 14 Pro 256G 暗紫色 |
| `price` | 价格 | 4800 |
| `condition` | 成色描述 | 9成新，有轻微使用痕迹 |
| `includes` | 包含配件 | 原装充电线、保护壳、钢化膜 |
| `notes` | 备注 | 不支持砍价，顺丰包邮 |
| `faq` | 常见问题 | 电池健康92%，无维修记录 |

### 如何获取商品 ID

1. 启动 Bot 后查看日志，会打印商品 ID
2. 或在闲鱼网页打开商品详情页，URL 中的数字就是 ID

### 示例

```json
{
  "123456789": {
    "title": "iPhone 14 Pro 256G 暗紫色",
    "price": "4800",
    "condition": "9成新，有轻微使用痕迹",
    "includes": "原装充电线、保护壳、钢化膜",
    "notes": "不支持砍价，顺丰包邮，支持验货",
    "faq": "电池健康度92%，无维修记录，已退出Apple ID"
  },
  "987654321": {
    "title": "罗技 MX Master 3S 鼠标",
    "price": "350",
    "condition": "全新未拆封",
    "notes": "顺丰到付，不议价"
  }
}
```

### 自动获取

如果某个商品 ID 没有在 `products.json` 中手动填写，Bot 会自动调用闲鱼 API 获取标题和价格。手动填写的信息优先级更高。
