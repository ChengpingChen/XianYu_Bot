# 产品知识库

## 使用方式

编辑 `products.json` 文件，按分类层级组织商品信息。

## 结构说明

`products.json` 使用**嵌套分类**结构，支持无限层级：

- 以分类名作为键名，内含 `items` 对象
- `items` 的 key 为商品 ID（闲鱼 itemId 或自定义 ID）
- 特殊键 `_说明` 和 `_通用` 为元数据（`_通用` 为店铺简介，会注入 AI prompt）

### 商品字段

| 字段 | 说明 | 必填 |
|------|------|------|
| `title` | 商品标题 | 是 |
| `content` | 完整商品描述（发货内容/AI参考） | 是 |
| `keywords` | 关键词数组（无闲鱼 ID 的商品用于 AI 匹配） | 否 |
| `price` | 价格（可选） | 否 |

### 如何获取商品 ID

1. 启动 Bot 后查看日志，会打印商品 ID
2. 或在闲鱼网页打开商品详情页，URL 中的数字就是 ID

### 自动获取

如果某个商品 ID 没有在 `products.json` 中手动填写，Bot 会自动调用闲鱼 API 获取标题和价格。手动填写的信息优先级更高。
